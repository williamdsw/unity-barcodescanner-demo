# Barcode Scanner Demo for Unity

Test project for [Unity Barcode Scanner plugin](https://github.com/kefniark/UnityBarcodeScanner).

## Running the project locally

* **Android Build** -> Download `.zip` file or `git clone` this repo from `master` branch.
* Make sure that you have version `2020.1.15f1` of Unity or more recent one.
* Open `Unity Hub` and import the project.
* Open the `Demo` scene, and hit the play button.
